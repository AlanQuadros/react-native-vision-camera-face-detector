## Getting Started

Install all dependencies with:
```bash
npm instal
# or
yarn
```

Then run the app in development mode:
```bash
npm run android
# or
npm run ios
# or
yarn android
# or
yarn ios
```

Or in production mode:
```bash
npm run android:prod
# or
npm run ios:prod
# or
yarn android:prod
# or
yarn ios:prod
```

## Cleaning

If, for some reason, you need to clean the project just run:
```bash
npm run prebuild:clean
# or
yarn prebuild:clean
```
